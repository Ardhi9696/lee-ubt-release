# Lee UBT EPS-TOPIK — APK Release

Repo ini hanya untuk distribusi file APK rilis (tanpa source code).

## Download APK
Silakan buka tab **Releases** dan unduh file APK versi terbaru.

## Catatan Instalasi
- Aktifkan: Settings → Security → **Install unknown apps**
- Jika update, unduh APK versi terbaru lalu install ulang (data biasanya tetap aman)

## Versi
- Lihat riwayat versi & changelog di halaman Releases.
